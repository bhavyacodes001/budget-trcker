# Income Expense Calculator

A modern, responsive web application for tracking income and expenses with a clean and intuitive interface.

## Features

- Add, edit, and delete income and expense entries
- Real-time calculation of total income, expenses, and net balance
- Filter entries by type (All, Income, Expense)
- Persistent data storage using localStorage
- Responsive design that works on both desktop and mobile devices
- Modern UI with smooth animations and transitions

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS)
- JavaScript (Vanilla)

## Getting Started

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. Start adding your income and expense entries!

## How to Use

1. **Adding an Entry**
   - Fill in the description and amount
   - Select the type (Income or Expense)
   - Click "Add Entry"

2. **Editing an Entry**
   - Click the edit (pencil) icon next to the entry
   - The form will be populated with the entry's data
   - Make your changes and click "Add Entry"

3. **Deleting an Entry**
   - Click the delete (trash) icon next to the entry
   - The entry will be removed immediately

4. **Filtering Entries**
   - Use the radio buttons at the top to filter entries by type
   - Choose between "All", "Income", or "Expense"

## Data Persistence

All your entries are automatically saved to your browser's localStorage, so they'll be there even after you close the browser.
